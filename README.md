# morals_in_LLM
Оценка морального выбора в ответах LLM

Для запуска кода маленького эксперимента зайти в папку проекта и использовать:
```
export GROQ_API_KEY="gsk_ваш_ключ"
python3 experiment.py
```

UPD: есть и большой эксперимент который включает в себя 110 запросов
Для запуска обновленного эксперимента: 
```
export GROQ_API_KEY="gsk_..."
python3 experiment_large.py
```
